# Як обрати шини для SUV та кросовера

## Чим відрізняються шини для SUV?

Кросовери та SUV важчі за легкові автомобілі і мають вищий центр ваги. Шини для них повинні:
- Витримувати більше навантаження (вищий індекс навантаження).
- Забезпечувати стабільність у поворотах при вищому центрі ваги.
- Мати посилену боковину для захисту від пошкоджень.

## Типи шин для SUV

### H/T (Highway Terrain) — шосейні
- Для міських кросоверів, що їздять переважно по асфальту.
- Тихі, комфортні, економічні.
- Підходять для: Toyota RAV4, Nissan Qashqai, Hyundai Tucson, Kia Sportage.

### A/T (All Terrain) — універсальні
- Компроміс між асфальтом та легким бездоріжжям.
- Агресивніший протектор, краще тримають на ґрунті та гравії.
- Трохи шумніші на асфальті.
- Підходять для: Mitsubishi Outlander, Subaru Forester, Toyota Land Cruiser Prado.

### M/T (Mud Terrain) — для бездоріжжя
- Великі блоки протектора, максимальне зчеплення в бруді та на бездоріжжі.
- Шумні на асфальті, швидше зношуються.
- Тільки для справжнього офроуду.
- Підходять для: Jeep Wrangler, Toyota Land Cruiser 70/200.

## Чи можна ставити легкові шини на кросовер?

**Категорично не рекомендується**, навіть якщо розмір збігається. Легкові шини:
- Мають нижчий індекс навантаження — не витримають вагу SUV.
- Тонша боковина — ризик грижі на нерівностях.
- Не розраховані на вищий центр ваги — гірша стабільність у поворотах.

Завжди обирайте шини, позначені **SUV**, **4x4** або з відповідним індексом навантаження.

## Особливості для повного приводу (AWD)

- Міняйте всі 4 шини одночасно — різниця в діаметрі пошкоджує трансмісію.
- Різниця глибини протектора між осями не повинна перевищувати 2-3 мм.
- Обирайте однакову модель на всі 4 колеса.

## Рекомендований розмір vs збільшений

Багато власників SUV хочуть поставити більші диски та ширші шини «для краси». Але для українських доріг:
- Штатний розмір з вищим профілем = кращий захист від ям.
- Збільшений диск з низьким профілем = ризик пошкодження дисків та шин.

Рекомендуємо дотримуватись штатних розмірів, особливо для щоденної їзди.
