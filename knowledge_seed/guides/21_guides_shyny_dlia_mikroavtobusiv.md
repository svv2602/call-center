# Шини для мікроавтобусів та фургонів — повний гайд

## Чим вантажні шини відрізняються від легкових?

Шини для мікроавтобусів (Mercedes Sprinter, VW Crafter, Renault Master, Fiat Ducato) та фургонів (VW Transporter, Ford Transit Custom, Renault Trafic) мають:

1. **Посилену конструкцію** — додаткові шари корду для вищого навантаження.
2. **Вищий індекс навантаження** — 100-121 (800-1450 кг на колесо).
3. **Маркування "C"** (Commercial) або **"LT"** (Light Truck) — на бічній стінці.
4. **Подвійний індекс навантаження** — наприклад, 107/105 (одинарне/здвоєне колесо).

## Маркування

### "C" (Commercial)
- Європейське маркування.
- Приклад: **215/65 R16 C 109/107T**
- C означає: посилена конструкція для комерційного транспорту.

### "LT" (Light Truck)
- Американське маркування (зустрічається рідше в Україні).
- Приклад: **LT 235/65 R16**

### Подвійний індекс навантаження
- 109/107 означає: **109** (1030 кг) при одинарному монтажі, **107** (975 кг) при здвоєному.

## Як обрати шини для мікроавтобуса?

### 1. Розмір
Перевірте рекомендований розмір:
- **Sprinter/Crafter:** зазвичай 235/65 R16 C або 225/75 R16 C.
- **Transporter/Transit Custom:** 215/65 R16 C або 205/65 R16 C.
- **Ducato/Master:** 225/70 R15 C або 235/65 R16 C.

### 2. Індекс навантаження
- Обирайте **не менший**, ніж рекомендований виробником авто.
- Для постійно завантаженого фургона — бажано на 1-2 позиції вище.

### 3. Сезон
- **Літні** — для перевезень у теплий сезон.
- **Зимові** — обов'язково для зимових перевезень (законодавчо для комерційного транспорту).
- **Всесезонні** — компроміс для фургонів з цілорічним використанням.

### 4. Тиск
- Комерційні шини працюють при **вищому тиску** (3.0-4.5 бар).
- Перевантаження при заниженому тиску — головна причина руйнування.
- **Завжди перевіряйте тиск** перед завантаженням.

## Топ шин для комерційного транспорту

### Преміум
- **Continental VanContact 4Season** — всесезонна з 3PMSF, великий ресурс (80 000+ км).
- **Michelin Agilis CrossClimate** — всесезонна, еталон для комерційного транспорту.
- **Goodyear EfficientGrip Cargo 2** — літня з великим ресурсом.

### Середній
- **Hankook Vantra LT** — хороше співвідношення ціна/ресурс.
- **Kumho PorTran** — бюджет-середній, популярний для малого бізнесу.
- **Firestone Vanhawk Multiseason** — всесезонна за помірну ціну.

### Бюджет
- **Laufenn X Fit Van** — бюджетна від Hankook для фургонів.
- **Petlas Full Grip PT925** — турецьке виробництво, хороша ціна.
- **Linglong GreenMax Van** — бюджетний варіант з непоганим ресурсом.

## Частота заміни та ротація

- **Ресурс:** 40 000–80 000 км (залежно від навантаження та бренду).
- **Ротація:** Кожні 10 000 км — обов'язково (передні зношуються швидше).
- **Тиск:** Перевірка **щотижня** (або перед кожним навантаженням).
- **Балансування:** При сезонній заміні та при появі вібрації.

## Типові помилки

1. **Встановлення легкових шин** замість C-класу — вони не витримають навантаження та лопнуть.
2. **Занижений тиск** при повному завантаженні — перегрів та руйнування каркасу.
3. **Ігнорування нерівномірного зносу** — на фургонах вказує на проблеми з підвіскою або перевантаження.
4. **Їзда на зношених зимових** — для комерційного транспорту мінімум 4 мм (не 1.6 мм!).
