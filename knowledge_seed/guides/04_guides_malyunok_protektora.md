# Типи малюнку протектора: який обрати?

## Симетричний протектор

Однаковий малюнок по всій ширині шини. Немає напрямку обертання та різниці між сторонами.

**Характеристики:**
- Тихий хід, добрий комфорт.
- Рівномірний знос протектора.
- Можна переставляти на будь-яке колесо.
- Найдоступніша ціна.

**Для кого:** спокійна міська їзда, невеликі пробіги, бюджетний сегмент.

**Приклади моделей:** Kumho Solus, Hankook Kinergy Eco, Lassa Greenways.

## Асиметричний протектор

Зовнішня та внутрішня частини мають різний малюнок. Зовнішня оптимізована для зчеплення на сухій дорозі, внутрішня — для водовідведення.

**Характеристики:**
- Відмінна керованість на сухій дорозі.
- Добре водовідведення.
- Стабільність на високій швидкості.
- Позначки OUTSIDE/INSIDE на боковині — ставити строго за маркуванням.

**Монтаж:** можна переставляти з лівого боку на правий і навпаки, головне — OUTSIDE назовні.

**Для кого:** активне водіння, шосе, середній та преміум сегмент.

**Приклади моделей:** Continental PremiumContact, Michelin Primacy, Bridgestone Turanza.

## Спрямований (V-подібний) протектор

V-подібні канавки, що сходяться до центру. Мають чіткий напрямок обертання.

**Характеристики:**
- Найкраще водовідведення серед усіх типів.
- Мінімальний ризик аквапланування.
- Ефективне самоочищення від снігу (для зимових моделей).
- Позначка ROTATION зі стрілкою на боковині.

**Обмеження:** не можна переставляти з лівого боку на правий без перебортування.

**Для кого:** регіони з частими дощами, зимові шини, водії, що цінують безпеку на мокрій дорозі.

**Приклади моделей:** Goodyear EfficientGrip, Nokian Hakkapeliitta (зимові), Pirelli Cinturato.

## Яка різниця на практиці?

| Параметр | Симетричний | Асиметричний | Спрямований |
|----------|-------------|--------------|-------------|
| Водовідведення | Добре | Дуже добре | Відмінне |
| Зчеплення на сухій дорозі | Добре | Відмінне | Дуже добре |
| Шум | Тихий | Середній | Середній |
| Зручність перестановки | Будь-яка позиція | Будь-яка (з урахуванням сторони) | Тільки тій самій стороні |
| Ціна | Найнижча | Середня–висока | Середня–висока |

## Порада

Для більшості водіїв в Україні **асиметричний протектор** — оптимальний вибір для літніх шин. Для зимових — **спрямований**, завдяки ефективному відведенню слякоті та самоочищенню від снігу.
