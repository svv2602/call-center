# Шини для електромобілів — повний гайд

## Чому електромобілям потрібні спеціальні шини?

Електромобілі (EV) суттєво відрізняються від авто з ДВЗ за характеристиками, що впливають на вибір шин:

1. **Вага на 20-30% більша** — батарея важить 300-700 кг. Потрібні шини з вищим індексом навантаження (XL/Reinforced).
2. **Миттєвий крутний момент** — електромотор видає 100% моменту з нуля. Звичайні шини зношуються швидше через пробуксовку при старті.
3. **Тиша** — без шуму ДВЗ шум від шин стає основним джерелом. Потрібні тихі шини.
4. **Запас ходу** — кожен відсоток опору кочення = втрата кілометрів. Потрібні енергоефективні шини.

## Топ шин для електромобілів

### Преміум
- **Michelin e.Primacy** — еталон енергоефективності. Опір кочення на 10-15% нижчий за конкурентів. +30 км запасу ходу. Тихий хід. Ціна: 3 500–5 500 грн (R17).
- **Continental EcoContact 6** — відмінне зчеплення на мокрому (EU Label A). Технологія ContiSilent для тиші. Ціна: 3 200–5 000 грн (R17).
- **Pirelli P Zero Elect** — спортивний варіант для потужних EV (Tesla Model S, Porsche Taycan). Ціна: 4 500–7 000 грн (R19).

### Середній
- **Hankook iON evo** — спеціальна EV-лінійка, хороший баланс. Ціна: 2 500–4 000 грн (R17).
- **Bridgestone Turanza T005** — комфорт та тиша, XL версії для EV. Ціна: 2 800–4 500 грн (R17).
- **Goodyear ElectricDrive 2** — для EV та гібридів, низький опір кочення. Ціна: 2 600–4 200 грн (R17).

### Бюджет
- **Sailun ERange** — одна з перших бюджетних EV-шин, прийнятна якість. Ціна: 1 600–2 500 грн (R17).
- **Linglong Green-Max HP010** — не спеціально для EV, але XL-версії підходять. Ціна: 1 400–2 200 грн (R17).

## Чи можна ставити звичайні шини на EV?

**Так, можна**, але з обмеженнями:
- **Обов'язково XL (Extra Load)** — стандартна шина не витримає ваги EV.
- **Ресурс буде менший** на 15-25% — через вагу та крутний момент.
- **Запас ходу зменшиться** на 3-8% — вищий опір кочення.
- **Шум буде більший** — без технологій шумозахисту.

Якщо бюджет обмежений — звичайна XL шина з низьким опором кочення (EU Label A-B) — прийнятний компроміс.

## Зимові шини для EV

Ті самі правила, що й для звичайних авто, плюс:
- **Обов'язково XL** — ще важливіше взимку (холодний тиск нижчий).
- Зимою запас ходу падає на 20-40% — вибирайте шини з мінімальним опором кочення.
- **Nokian Hakkapeliitta R5 EV** — спеціальна зимова для EV.
- **Continental VikingContact 7** — XL версії підходять для EV.

## Ресурс шин на EV

Через вагу та момент шини на EV зношуються на **15-30% швидше**, ніж на аналогічних ДВЗ-авто. Ротація кожні 8 000-10 000 км (а не 10 000-12 000) — обов'язкова для рівномірного зносу.
