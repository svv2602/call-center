# Ротація шин — схеми перестановки та частота

## Що таке ротація шин?

Ротація — це перестановка шин між позиціями (перед/зад, ліва/права) для рівномірного зносу. На передньопривідних авто передні шини зношуються в 1.5-2 рази швидше за задні.

## Коли робити ротацію?

- **Кожні 10 000–12 000 км** — оптимальний інтервал.
- **При сезонній заміні** — найзручніший момент, шини вже знімають.
- **При нерівномірному зносі** — якщо помітили різницю в глибині протектора між осями.
- **Після першої 1000 км** нових шин — деякі виробники рекомендують.

## Схеми ротації

### Передньопривідні авто (FWD)
```
Ненаправлений протектор:
  Перед ліва → Зад права
  Перед права → Зад ліва
  Зад ліва → Перед ліва
  Зад права → Перед права
  (Хрестова схема: передні назад навхрест, задні вперед прямо)
```

### Задньопривідні авто (RWD)
```
Ненаправлений протектор:
  Зад ліва → Перед права
  Зад права → Перед ліва
  Перед ліва → Зад ліва
  Перед права → Зад права
  (Зворотна хрестова: задні вперед навхрест, передні назад прямо)
```

### Повнопривідні авто (AWD/4WD)
```
  Перед ліва → Зад права
  Перед права → Зад ліва
  Зад ліва → Перед права
  Зад права → Перед ліва
  (Подвійна хрестова: усі навхрест)
```

### Направлений протектор (будь-який привід)
```
  Перед ліва ↔ Зад ліва
  Перед права ↔ Зад права
  (Тільки перед-зад, без зміни сторони — стрілка на бічній стінці показує напрямок обертання)
```

### Асиметричний протектор
- Якщо **не направлений** — ротація як звичайний (хрестова схема).
- Маркування "OUTSIDE" завжди має бути зовні.

## Коли ротація НЕ потрібна або неможлива?

- **Різні розміри перед/зад** (наприклад, BMW, Mercedes AMG) — ротація неможлива.
- **Різні шини перед/зад** — не міняти місцями.
- **RunFlat та звичайні на одному авто** — не комбінувати.
- **Шина має грижу або пошкодження** — замінити, а не переставляти.

## Результати регулярної ротації

- **+20-30% ресурсу** шин (рівномірний знос).
- Збалансована керованість та гальмування.
- Можливість замінити всі 4 шини одночасно (а не парами).
- Економія до 2000-3000 грн на комплект за рахунок продовженого ресурсу.

## Ротація + балансування

При кожній ротації рекомендується також **балансувати** колеса. Більшість шиномонтажних станцій включають балансування у вартість ротації або пропонують знижку при комплексному обслуговуванні.
