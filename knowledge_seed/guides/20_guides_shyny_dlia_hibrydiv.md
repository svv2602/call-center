# Шини для гібридів — особливості вибору

## Чим гібрид відрізняється від звичайного авто?

Гібриди (HEV, PHEV) мають:
- **Додаткову вагу** — електромотор + батарея додають 50-200 кг (HEV) або 150-400 кг (PHEV).
- **Електричний режим** — при русі на електриці шум шин = основний шум в салоні.
- **Рекуперативне гальмування** — менше навантаження на шини при гальмуванні.
- **Режим "EV"** — для максимального запасу ходу на електриці потрібен низький опір кочення.

## Які шини потрібні?

### Обов'язково
- **XL (Extra Load)** — для PHEV (plug-in гібрид) з великою батареєю.
- **Правильний індекс навантаження** — дивіться рекомендації виробника авто.

### Рекомендовано
- **Низький опір кочення** (EU Label A-B) — для максимального електричного запасу ходу.
- **Знижений рівень шуму** — комфорт при русі на електриці.

### Не обов'язково
- Спеціальне EV-маркування — для м'яких гібридів (HEV) з невеликою батареєю звичайні якісні шини підходять.

## Рекомендації за типом гібрида

### Mild Hybrid (MHEV) — Toyota Corolla, Hyundai Tucson, Kia Sportage
- Різниця у вазі мінімальна (+20-50 кг).
- **Звичайні шини підходять** — не потрібен XL.
- Рекомендація: будь-які якісні шини правильного розміру.

### Full Hybrid (HEV) — Toyota RAV4 Hybrid, Honda CR-V Hybrid
- Додаткова вага +100-150 кг.
- XL бажано, але не критично для більшості моделей.
- Рекомендація: шини з низьким опором кочення (EU Label A-B).

### Plug-in Hybrid (PHEV) — Mitsubishi Outlander PHEV, BMW X5 xDrive45e
- Додаткова вага +200-400 кг (велика батарея).
- **XL обов'язково** — перевірте рекомендації виробника авто.
- Рекомендація: EV-шини або XL з низьким опором кочення.

## Топ шин для гібридів

### Універсальні (для HEV та PHEV)
- **Michelin e.Primacy** — найкращий опір кочення, максимальний ресурс.
- **Continental EcoContact 6** — відмінне зчеплення на мокрому + енергоефективність.
- **Bridgestone Turanza Eco** — спеціально для гібридів та EV.

### Бюджетні
- **Hankook Kinergy Eco 2** — низький опір кочення за помірну ціну.
- **Kumho Ecowing ES31** — бюджетний варіант з EU Label B.

## Зимові шини для гібридів

Зимою гібриди втрачають значну частину електричного запасу ходу. Вибирайте зимові шини з:
- **Мінімальним опором кочення** — щоб зберегти електричний запас.
- **XL для PHEV** — обов'язково.
- **3PMSF маркування** — для безпеки на снігу.

Конкретні моделі: ті самі, що й для звичайних авто відповідного розміру, але обов'язково XL для PHEV.

## Ресурс шин на гібридах

- **MHEV:** Ресурс такий самий, як на ДВЗ.
- **HEV:** Ресурс зменшується на 5-10% (трохи більша вага + регенеративне гальмування менше зношує гальмівні колодки, але більше — шини).
- **PHEV:** Ресурс зменшується на 10-20% (значна додаткова вага). Ротація кожні 8 000-10 000 км.
