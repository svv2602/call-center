# Замовлення для юридичних осіб та ФОП

## Що ми пропонуємо

Ми працюємо з юридичними особами та ФОП: автопарки, таксі, логістичні компанії, СТО, автодилери.

## Переваги для юридичних осіб

- **Безготівковий розрахунок** — оплата за рахунком-фактурою.
- **Повний пакет документів** — видаткова накладна, рахунок-фактура, акт виконаних робіт.
- **Можливість відстрочки платежу** — для постійних клієнтів, за домовленістю.
- **Оптові ціни** — знижки від 5 комплектів (20 шин).
- **Персональний менеджер** — для регулярних замовлень.

## Як оформити замовлення

### Крок 1: Надайте реквізити
- Назва підприємства.
- Код ЄДРПОУ.
- Юридична адреса.
- ІПН (якщо платник ПДВ).
- Контактна особа та телефон.

### Крок 2: Отримайте рахунок
Ми сформуємо рахунок-фактуру з усіма реквізитами. Надсилаємо на email.

### Крок 3: Оплата
- Оплата за реквізитами з рахунку.
- Термін оплати: передоплата або відстрочка (за домовленістю).

### Крок 4: Доставка
- Стандартна доставка Новою Поштою або самовивіз.
- Для великих замовлень — можлива доставка власним транспортом.

### Крок 5: Документи
Після відвантаження надсилаємо:
- Видаткову накладну (оригінал з посилкою або поштою).
- Податкову накладну (для платників ПДВ).

## Оптові замовлення

| Кількість | Знижка |
|-----------|--------|
| 5-9 комплектів (20-36 шин) | Індивідуальна ціна |
| 10+ комплектів (40+ шин) | Спеціальна оптова ціна |
| Регулярні поставки | Договірна ціна + відстрочка |

Для отримання оптової ціни зверніться до оператора — ми підготуємо індивідуальну пропозицію.

## Шини для автопарків

Для компаній з автопарками ми пропонуємо:
- Підбір оптимальних шин за критерієм вартість/км (TCO).
- Рекомендації з обслуговування для максимального ресурсу.
- Графік сезонних замін з резервуванням потрібних розмірів.
- Оперативну заміну при пошкодженні (наявність ходових розмірів на складі).
