# Як обрати колісні диски

## Крок 1: Визначте параметри

Кожен автомобіль має рекомендовані виробником параметри дисків. Їх можна знайти:
- У інструкції до автомобіля
- На наклейці на стійці водійських дверей
- У нашому каталозі — введіть марку та модель авто

### Основні параметри
- **Діаметр (R)** — R15, R16, R17 тощо
- **Ширина** — у дюймах (6J, 7J, 7.5J)
- **Розболтовка (PCD)** — кількість болтів та діаметр кола (5×112, 4×100)
- **Вилет (ET)** — відстань від площини кріплення до осі симетрії диска (мм)
- **Центральний отвір (DIA)** — діаметр центрального отвору (мм)

## Крок 2: Оберіть тип диска

- **Штамповані** — для бюджету або зимового комплекту
- **Литі** — оптимальне співвідношення ціна/якість/дизайн
- **Ковані** — максимальна якість та мінімальна вага

## Крок 3: Перевірте сумісність із шинами

Ширина диска має відповідати ширині шини:
| Ширина диска | Рекомендована ширина шини |
|-------------|-------------------------|
| 6J | 185–205 мм |
| 6.5J | 195–215 мм |
| 7J | 205–225 мм |
| 7.5J | 215–235 мм |
| 8J | 225–245 мм |

## Крок 4: Зверніть увагу на деталі

- **Кріплення** — болти або гайки (залежить від авто). При зміні типу диска може потрібна заміна кріплення
- **Ковпаки** — для штампованих дисків оберіть розмір ковпака за діаметром диска
- **Центрувальні кільця** — якщо DIA диска більший за DIA маточини, потрібні кільця для точного центрування

## Часті помилки

- Обрати диски з неправильною розболтовкою — не стануть на авто
- Ігнорувати вилет (ET) — може впливати на керованість та зношування підвіски
- Ставити занадто широкі диски — шини будуть виступати за арки
