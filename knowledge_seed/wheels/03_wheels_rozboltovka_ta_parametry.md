# Розболтовка та параметри дисків

## Розболтовка (PCD)

PCD (Pitch Circle Diameter) — це кількість кріпильних отворів та діаметр кола, на якому вони розташовані.

### Формат запису
**Кількість отворів × діаметр кола в мм**

Наприклад: 5×112 означає 5 болтів на колі діаметром 112 мм.

### Популярні розболтовки по марках
| Розболтовка | Марки автомобілів |
|------------|-------------------|
| 4×98 | Fiat, Alfa Romeo, Lancia |
| 4×100 | Honda, Toyota, Hyundai, Kia, Mazda, VW (старі) |
| 4×108 | Ford, Peugeot, Citroën |
| 5×100 | Subaru, Toyota, VW Golf IV–V, Skoda |
| 5×108 | Ford (нові), Volvo, Land Rover |
| 5×112 | VW, Audi, Skoda, Mercedes, Seat |
| 5×114.3 | Toyota, Honda, Hyundai, Kia, Mazda, Nissan, Mitsubishi |
| 5×120 | BMW, Land Rover |
| 5×130 | Porsche, VW Touareg |

**Важливо:** Розболтовка має збігатися точно. Диски 5×112 не підходять на авто з 5×114.3, навіть якщо різниця здається малою.

## Вилет (ET)

Вилет — відстань у міліметрах від площини кріплення диска до його геометричного центру.

- **Позитивний ET** (найпоширеніший) — площина кріплення зміщена назовні від центру
- **Нульовий ET** — площина кріплення збігається з центром
- **Негативний ET** — площина кріплення зміщена всередину

### Допустимі відхилення
- Рекомендовано: ±5 мм від заводського значення
- Більше — може вплинути на підвіску, рульове керування, зношування підшипників

## Центральний отвір (DIA)

DIA — діаметр центрального отвору диска в міліметрах.

- Має дорівнювати або бути **більшим** за діаметр маточини авто
- Якщо DIA диска більший — використовуйте **центрувальні кільця** (пластикові або алюмінієві)
- Якщо DIA диска менший — диск **не підійде**

### Приклади DIA по марках
| Марка | DIA (мм) |
|-------|---------|
| VW / Audi / Skoda | 57.1 |
| BMW | 72.6 |
| Mercedes | 66.6 |
| Toyota | 60.1 |
| Hyundai / Kia | 67.1 |
| Ford | 63.4 |
