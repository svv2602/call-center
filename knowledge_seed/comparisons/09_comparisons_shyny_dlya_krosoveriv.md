# Порівняння шин для кросоверів: H/T vs A/T

## Два основних типи

Для кросоверів та SUV є два основних типи шин:
- **H/T (Highway Terrain)** — шосейні, для асфальту.
- **A/T (All Terrain)** — універсальні, для асфальту та легкого бездоріжжя.

## Детальне порівняння

| Параметр | H/T (шосейні) | A/T (універсальні) |
|----------|---------------|---------------------|
| Асфальт (суха дорога) | Відмінно | Добре |
| Асфальт (мокра дорога) | Відмінно | Добре |
| Гравій та ґрунтові дороги | Задовільно | Дуже добре |
| Бруд та м'який ґрунт | Погано | Добре |
| Сніг (неочищені дороги) | Задовільно | Дуже добре |
| Шум на асфальті | Тихі | Помітно шумніші |
| Комфорт | Високий | Середній |
| Ресурс протектора | 50 000–70 000 км | 40 000–60 000 км |
| Витрата палива | Нижча | На 5-10% вища |
| Ціна | Стандартна | На 15-25% дорожча |

## Популярні H/T моделі для кросоверів

### Continental CrossContact RX
- Лідер у гальмуванні на мокрій дорозі.
- Дуже тихі для SUV-шин.
- Підходить для: міські кросовери (Tucson, Sportage, RAV4).

### Michelin Primacy SUV+
- Максимальний комфорт та довговічність.
- Найнижча витрата палива у класі.
- Підходить для: преміум кросовери та великі пробіги.

### Hankook Ventus S1 evo3 SUV
- Спортивна керованість для SUV.
- Відмінне співвідношення ціна/якість.
- Підходить для: динамічне водіння на кросовері.

## Популярні A/T моделі

### Continental CrossContact ATR
- Найкращий баланс асфальт/бездоріжжя.
- Помірний шум для A/T.
- Підходить для: SUV, які іноді з'їжджають з асфальту.

### BFGoodrich All-Terrain T/A KO2
- Легендарна модель для бездоріжжя.
- Посилена боковина (Kevlar).
- Підходить для: серйозне бездоріжжя та позашляхові подорожі.

### Yokohama Geolandar A/T G015
- Тихіший за більшість A/T.
- Добре тримає на мокрій дорозі.
- Підходить для: компроміс між комфортом та прохідністю.

## Що обрати?

### Обирайте H/T, якщо:
- 90%+ часу їздите по асфальту.
- Цінуєте тишу та комфорт.
- Хочете мінімальну витрату палива.
- Ваш кросовер — міський (Tucson, Sportage, Qashqai, RAV4).

### Обирайте A/T, якщо:
- Регулярно їздите ґрунтовими дорогами (дача, село, рибалка).
- Взимку ваші дороги часто не чистять.
- Хочете підвищений захист від проколів (товстіша боковина).
- Ваш SUV — рамний або з серйозним позашляховим потенціалом.

## Важливо

Для більшості українських кросоверів (міське використання) **H/T — оптимальний вибір**. A/T виправдані тільки якщо ви реально їздите по бездоріжжю. Агресивний протектор A/T на міському кросовері — це зайвий шум, витрата палива та швидший знос без реальної потреби.
