# Шини для мікроавтобусів та легких вантажівок

## Маркування

Шини для комерційного транспорту мають спеціальне маркування:

### Маркування «C» (Commercial)
- Вказується в розмірі: **205/65 R16C**
- Означає посилену конструкцію для підвищених навантажень
- Більша кількість шарів корду (6–10 замість 2–4 у легкових)

### Маркування «LT» (Light Truck)
- Американська система: **LT245/75R16**
- Аналог європейського «C»

### Подвійний індекс навантаження
- Наприклад: 107/105T — перше число для одинарного колеса, друге для здвоєного

## Типи шин для комерційного транспорту

### Шосейні (Highway)
- Для переважно асфальтового покриття
- Низький рівень шуму та хороша паливна ефективність
- Підходять для: розвізних фургонів, пасажирських мікроавтобусів

### Універсальні (All-Position)
- Для будь-якої осі (кермова, ведуча, причіпна)
- Збалансовані характеристики
- Підходять для: мікроавтобусів, легких вантажівок зі змішаним циклом

### Зимові комерційні
- Маркування M+S та 3PMSF (сніжинка)
- Посилена конструкція + зимова гумова суміш
- Обов'язкові для вантажних перевезень взимку

## Особливості експлуатації

- **Тиск** — вищий, ніж у легкових (3.0–5.5 атм залежно від навантаження). Перевіряйте щотижня
- **Навантаження** — не перевищуйте максимальне навантаження на шину. Перевантаження — головна причина передчасного зносу
- **Ротація** — кожні 10 000 км для рівномірного зносу
- **Вік** — комерційні шини експлуатуються інтенсивніше, замінюйте не пізніше 6 років незалежно від залишку протектора
