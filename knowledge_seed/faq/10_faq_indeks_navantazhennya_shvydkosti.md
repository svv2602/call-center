# Що означають індекс навантаження та індекс швидкості?

## Індекс навантаження

Це число на боковині шини (наприклад, **91** у маркуванні 205/55 R16 **91**V), що позначає максимальну вагу, яку витримує одна шина.

### Таблиця основних індексів навантаження

| Індекс | Макс. навантаження |
|--------|-------------------|
| 75 | 387 кг |
| 80 | 450 кг |
| 85 | 515 кг |
| 88 | 560 кг |
| 91 | 615 кг |
| 94 | 670 кг |
| 97 | 730 кг |
| 100 | 800 кг |
| 104 | 900 кг |
| 110 | 1060 кг |

### Як розрахувати?

Помножте індекс навантаження одної шини на 4 — це максимальна вага завантаженого автомобіля. Наприклад: 91 × 4 = 2460 кг.

**Важливо:** не ставте шини з індексом навантаження нижчим, ніж рекомендує виробник авто. Вищий — можна.

## Індекс швидкості

Це літера після індексу навантаження (наприклад, **V** у маркуванні 205/55 R16 91**V**), що позначає максимальну допустиму швидкість.

### Таблиця індексів швидкості

| Літера | Макс. швидкість |
|--------|----------------|
| Q | 160 км/год |
| R | 170 км/год |
| S | 180 км/год |
| T | 190 км/год |
| H | 210 км/год |
| V | 240 км/год |
| W | 270 км/год |
| Y | 300 км/год |

**Важливо:** не ставте шини з індексом швидкості нижчим, ніж рекомендує виробник авто. Для зимових шин допускається на один рівень нижче (наприклад, H замість V).

## Позначення XL (Extra Load)

Шини з маркуванням XL мають підвищений індекс навантаження (+3 одиниці) завдяки посиленій конструкції. Вони трохи важчі та жорсткіші, але витримують більшу вагу.
