# Шини для електромобілів та гібридів

## Чим відрізняються шини для електромобілів

Електромобілі (EV) мають особливості, які висувають специфічні вимоги до шин:

### 1. Більша вага
Батарея додає 300–700 кг до маси автомобіля. Шини мають витримувати підвищене навантаження — потрібен вищий індекс навантаження.

### 2. Миттєвий крутний момент
Електромотор видає максимальний крутний момент з місця. Звичайні шини зношуються на 20–30% швидше через інтенсивне прискорення.

### 3. Низький рівень шуму
Без шуму двигуна шум від шин стає основним джерелом звуку в салоні. Для EV розробляють шини зі зниженим рівнем шуму (технології поролонового шару всередині шини).

### 4. Запас ходу
Опір коченню прямо впливає на запас ходу. Шини з низьким опором коченню додають до 10% запасу.

## Спеціальні маркування

Виробники шин позначають моделі для електромобілів спеціальними маркуваннями:
- **Michelin** — маркування «Acoustic» (шумозниження) та «EV Ready»
- **Continental** — маркування «ContiSilent» та «EV»
- **Hankook** — серія iON (спеціально для EV)
- **Pirelli** — маркування «Elect»
- **Bridgestone** — серія Turanza EV

## Чи можна ставити звичайні шини на електромобіль

Так, але є нюанси:
- Перевірте **індекс навантаження** — він має бути не менший за рекомендований виробником авто
- Звичайні шини зношуються швидше через підвищений крутний момент
- Запас ходу може зменшитися на 5–10%
- Рівень шуму буде вищим

## Ціни

Спеціалізовані шини для EV коштують на 10–20% дорожче звичайних аналогів. Але довший строк служби та економія на запасі ходу компенсують різницю.
