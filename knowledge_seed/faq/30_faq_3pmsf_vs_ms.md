# 3PMSF vs M+S — різниця маркувань зимових шин

## Два маркування — різна надійність

На бічній стінці шин можна побачити два типи зимових маркувань. Вони означають РІЗНІ речі.

## M+S (Mud and Snow)

### Що означає?
- **Mud and Snow** — "бруд та сніг".
- Самодекларація виробника — **не потребує тестування**.
- Виробник сам вирішує, чи відповідає шина критеріям M+S.

### Де зустрічається?
- На **літніх SUV-шинах** — багато AT-шин мають M+S, хоча вони непридатні для зими.
- На **всесезонних шинах** — як мінімальне маркування.
- На **зимових шинах** — разом із 3PMSF.
- На **літніх шинах** з агресивним протектором.

### Чи гарантує зимову безпеку?
**НІ.** M+S — це лише заява виробника, без підтвердження тестами. Шина з M+S може мати абсолютно непридатну для зими гумову суміш.

## 3PMSF (Three Peak Mountain Snowflake)

### Що означає?
- **Сніжинка в трикутнику** (три піки гори) — ❄️.
- Означає, що шина **пройшла стандартизовані тести** на гальмування на снігу.
- Тест: шина повинна гальмувати на снігу не гірше за еталонну зимову шину.

### Де зустрічається?
- На **зимових шинах** — обов'язково.
- На **якісних всесезонних шинах** — CrossClimate, Vector 4Seasons, Kinergy 4S2 тощо.
- На деяких **AT-шинах** — BFGoodrich KO2, Cooper AT3 4S, General AT3.

### Чи гарантує зимову безпеку?
**ТАК** — для снігу. Шина з 3PMSF пройшла реальний тест та має характеристики, підтверджені незалежною лабораторією.

**АЛЕ:** 3PMSF тестує лише **сніг**, не **лід**. Зчеплення на льоду залежить від гумової суміші та ламелей — це не входить у тест.

## Порівняння

| Критерій | M+S | 3PMSF |
|----------|-----|-------|
| **Тестування** | Ні (самодекларація) | Так (стандарт UNECE R117) |
| **Гарантія безпеки на снігу** | Ні | Так |
| **Визнання як "зимова шина"** | Не всюди | Так (ЄС, Україна) |
| **Достатньо для зими?** | Ні | Для помірних зим — так |
| **Вартість сертифікації** | 0 | ~10 000 EUR за модель |

## Практичні рекомендації

### Для зими в Україні
- **Обов'язково 3PMSF** — не купуйте зимові шини без сніжинки.
- **M+S без 3PMSF** — НЕ є зимовою шиною, навіть якщо продавець каже інакше.

### Для всесезонних шин
- Обирайте тільки з **3PMSF** (Michelin CrossClimate, Goodyear Vector 4Seasons, тощо).
- Всесезонна з лише M+S — це фактично літня шина з агресивним протектором.

### Для SUV/AT-шин
- AT-шина з **3PMSF** (BFGoodrich KO2, Cooper AT3 4S) — можна використовувати як зимову в помірних умовах.
- AT-шина з лише **M+S** — не підходить для зими.
