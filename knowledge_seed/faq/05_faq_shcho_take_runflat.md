# Що таке шини Run Flat?

## Принцип роботи

Run Flat (RF, ROF, RFT) — це технологія посилених боковин шини, яка дозволяє продовжувати рух навіть після повної втрати тиску. Посилені боковини тримають вагу автомобіля замість повітря.

## Скільки можна їхати на спущеній шині Run Flat?

- **Відстань**: до 80 км.
- **Швидкість**: не більше 80 км/год.
- Цього зазвичай достатньо, щоб дістатися до найближчого шиномонтажу.

## Обов'язкова умова

Шини Run Flat можна використовувати **тільки на автомобілях з системою контролю тиску в шинах (TPMS)**. Без TPMS водій може не помітити прокол і продовжити їзду на високій швидкості, що небезпечно.

## Переваги

- Не потрібно зупинятися на узбіччі при проколі (безпека).
- Не потрібна запаска — економія місця в багажнику.
- Стабільність автомобіля зберігається навіть при проколі.

## Недоліки

- **Дорожчі** — на 20-40% порівняно зі звичайними шинами тієї ж моделі.
- **Жорсткіший хід** — посилені боковини гірше поглинають нерівності.
- **Обмежений вибір моделей** — не всі розміри доступні в Run Flat.
- **Не підлягають ремонту** після їзди в спущеному стані — тільки заміна.

## Кому підходять?

Run Flat — оптимальний вибір для міської їзди та автомобілів без запаски. Якщо ви часто їздите трасою на великі відстані та цінуєте комфорт — звичайні шини + запаска можуть бути кращим варіантом.
