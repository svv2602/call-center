# Michelin — профіль бренду

## Загальна інформація

- **Країна:** Франція (Клермон-Ферран).
- **Рік заснування:** 1889.
- **Позиція на ринку:** Один із трьох найбільших виробників шин у світі.
- **Ціновий сегмент:** Преміум.
- **Конвеєрні поставки:** Porsche, BMW, Mercedes-Benz, Audi, Tesla, Toyota.

## Ключові технології

- **EverGrip** — протектор зберігає зчеплення на мокрій дорозі навіть при значному зносі. Приховані канавки відкриваються по мірі стирання.
- **Helio+** — гумова суміш на основі соняшникової олії для кращої еластичності при низьких температурах.
- **MaxTouch** — оптимізований контактний відбиток для рівномірного розподілу навантаження та збільшення ресурсу.

## Популярні лінійки

### Літні
- **Primacy 4+** — комфорт, тиша, максимальний ресурс. Для щоденної їзди.
- **Pilot Sport 5** — спортивна керованість. Для активного водіння.
- **e.Primacy** — максимальна енергоефективність. Для гібридів та електромобілів.

### Зимові
- **Alpin 6** — фрикційні, відмінний ресурс та зчеплення на снігу.
- **X-Ice Snow** — для суворих зим, максимальне зчеплення на льоду серед фрикційних.

### Всесезонні
- **CrossClimate 2** — еталон серед всесезонних, сертифікація 3PMSF.

## Сильні сторони

- Найдовший ресурс серед преміум-брендів (50 000–70 000 км для літніх).
- Стабільні характеристики навіть при зносі протектора.
- Низький опір кочення — економія палива.
- Програма Michelin Promise — безкоштовна заміна при пошкодженні протягом 12 місяців.

## Слабкі сторони

- Найвища ціна серед преміуму.
- Зимові моделі поступаються Nokian та Continental на чистому льоду.

## Для кого?

Водії, які цінують довговічність, комфорт та готові інвестувати у якість. Оптимальний вибір для великих пробігів — ціна окупається ресурсом.
