# Goodyear — профіль бренду

## Загальна інформація

- **Країна:** США (Акрон, Огайо).
- **Рік заснування:** 1898.
- **Позиція на ринку:** Третій за величиною виробник шин у світі. Легендарний бренд, шини Goodyear були на Місяці (Apollo 14).
- **Ціновий сегмент:** Преміум.
- **Конвеєрні поставки:** Ford, Jeep, Tesla, Chrysler, BMW.

## Ключові технології

- **ActiveBraking** — збільшення контактної площі при гальмуванні. Блоки протектора деформуються та «чіпляються» за дорогу.
- **SoundComfort** — шар поролону всередині шини для поглинання шуму.
- **Electric Drive Technology** — спеціальні рішення для електромобілів (низький опір кочення, тиша, підвищене навантаження).

## Популярні лінійки

### Літні
- **Eagle F1 Asymmetric 6** — спортивні, агресивне зчеплення та точне кермування.
- **EfficientGrip Performance 2** — рекордний ресурс (до 70 000 км), економія палива.
- **Eagle F1 SuperSport** — трекова шина для максимальної продуктивності.

### Зимові
- **UltraGrip Performance 3** — фрикційні, збалансовані характеристики.
- **UltraGrip Ice 2+** — для суворих зим, покращене зчеплення на льоду.
- **UltraGrip Arctic 2** — шиповані, для максимального зчеплення на льоду.

### Всесезонні
- **Vector 4Seasons Gen-3** — один із найкращих всесезонних, сертифікація 3PMSF.

## Сильні сторони

- Рекордний ресурс літніх шин (EfficientGrip Performance — до 70 000 км).
- Відмінна паливна ефективність (клас A за EU Label).
- Агресивна спортивна лінійка Eagle F1.
- Великий вибір всесезонних моделей.

## Слабкі сторони

- Зимові моделі поступаються Nokian та Continental.
- Деякі моделі шумніші за конкурентів (особливо Eagle F1).
- Менш розповсюджений на конвеєрах європейських авто.

## Бюджетні суббренди

Goodyear володіє брендами:
- **Sava** (Словенія) — тихі, комфортні бюджетні шини.
- **Debica** (Польща) — доступний бренд для економних водіїв.
- **Fulda** (Німеччина) — середній сегмент, популярний у Європі.
- **Dunlop** — середній-преміум сегмент.

## Для кого?

Водії з великими пробігами, яким важливий ресурс та економія палива. Ідеальний для тих, хто рахує вартість кілометра пробігу.
