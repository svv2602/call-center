# Bridgestone — профіль бренду

## Загальна інформація

- **Країна:** Японія (Токіо).
- **Рік заснування:** 1931.
- **Позиція на ринку:** Найбільший виробник шин у світі за обсягом продажів.
- **Ціновий сегмент:** Преміум.
- **Конвеєрні поставки:** Toyota, Honda, Nissan, Subaru, Lexus, Ferrari, Porsche.

## Ключові технології

- **NanoPro-Tech** — контроль взаємодії полімерів на молекулярному рівні. Зменшує опір кочення без втрати зчеплення.
- **ENLITEN** — полегшена конструкція (менше вага на 20%), зниження опору кочення для електромобілів.
- **3D Sipe Technology** — тривимірні ламелі, що блокуються при гальмуванні. Забезпечують стабільність блоків протектора.

## Популярні лінійки

### Літні
- **Turanza T005** — комфорт та безпека на мокрій дорозі. Для щоденної їзди.
- **Turanza 6** — нове покоління, покращені характеристики у всіх категоріях.
- **Potenza Sport** — максимальне зчеплення для спортивного водіння.

### Зимові
- **Blizzak LM005** — фрикційні, один із лідерів на мокрій зимовій дорозі.
- **Blizzak Ice** — для суворих зим, покращене зчеплення на льоду.

### Для SUV
- **Dueler H/T** — шосейні для кросоверів, комфорт та ресурс.
- **Dueler A/T** — універсальні, для легкого бездоріжжя.

## Сильні сторони

- Японська якість виготовлення — мінімум виробничого браку.
- Відмінні характеристики на мокрій дорозі (Turanza, Blizzak).
- Широкий асортимент для всіх типів автомобілів.
- Тихий та комфортний хід (особливо Turanza).

## Слабкі сторони

- Ресурс літніх шин трохи нижчий за Michelin та Goodyear.
- Ціна на рівні Continental, але з меншою маркетинговою впізнаваністю в Україні.
- Менше інноваційних програм для споживачів (порівняно з Michelin Promise).

## Бюджетний суббренд

Bridgestone володіє брендом **Lassa** (Туреччина) — популярні бюджетні шини з хорошим контролем якості.

## Для кого?

Водії, які цінують тишу, комфорт та надійність. Ідеальний вибір для японських автомобілів (Toyota, Honda, Mazda) — найкраща сумісність із підвіскою.
