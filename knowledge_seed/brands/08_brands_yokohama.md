# Yokohama — профіль бренду

## Загальна інформація

- **Країна:** Японія (Токіо).
- **Рік заснування:** 1917.
- **Позиція на ринку:** Один із найстаріших японських шинних виробників. Сьомий за величиною у світі.
- **Ціновий сегмент:** Середній-преміум.
- **Конвеєрні поставки:** Porsche, Mercedes-Benz, Toyota, Lexus, Mazda, Subaru.

## Ключові технології

- **BluEarth** — екологічна технологія. Перший виробник, який використав рослинну олію (апельсинова олія) у гумовій суміші для зниження опору кочення.
- **Advan** — спортивна платформа з технологіями з автоспорту (WRC, Super GT).
- **Geolandar** — спеціалізована платформа для позашляхових шин.

## Популярні лінійки

### Літні
- **BluEarth-GT AE51** — економічна їзда, максимальний ресурс, низький шум.
- **Advan Sport V107** — спортивна лінійка, конвеєр Porsche.
- **Advan Fleva V701** — доступна спортивна шина, гарне зчеплення.

### Зимові
- **BluEarth Winter V906** — фрикційні, добрий баланс характеристик.
- **iceGUARD iG60** — для суворих зим, зчеплення на льоду.

### Для SUV
- **Geolandar A/T G015** — легенда серед A/T, тихіша за конкурентів.
- **Geolandar X-CV** — спортивні шосейні для великих SUV.
- **Geolandar H/T G056** — комфортні шосейні для кросоверів.

## Сильні сторони

- Відмінне співвідношення ціна/якість (дешевший за Pirelli при близькій якості).
- Лідер у екологічних технологіях — найнижчий опір кочення.
- Сильна лінійка для SUV (Geolandar).
- Конвеєрні поставки Porsche — підтвердження спортивних характеристик.

## Слабкі сторони

- Менший асортимент розмірів в Україні порівняно з Continental або Michelin.
- Зимові шини — середній рівень (не спеціалізація бренду).
- Менша впізнаваність бренду серед українських водіїв.

## Для кого?

Власники японських автомобілів (Toyota, Mazda, Subaru), SUV та кросоверів. Ті, хто шукає преміальну якість за помірну ціну, особливо в сегменті SUV-шин.
