# Pirelli — профіль бренду

## Загальна інформація

- **Країна:** Італія (Мілан).
- **Рік заснування:** 1872.
- **Позиція на ринку:** П'ятий за величиною виробник шин. Офіційний ексклюзивний постачальник Формули-1 з 2011 року.
- **Ціновий сегмент:** Преміум та ультра-преміум.
- **Конвеєрні поставки:** Ferrari, Lamborghini, Maserati, BMW, Mercedes-AMG, Porsche, McLaren.

## Ключові технології

- **P Zero** — легендарна лінійка спортивних шин, технології з Формули-1.
- **Elect** — маркування для шин, адаптованих під електромобілі (низький шум, низький опір кочення, підвищене навантаження).
- **Seal Inside** — самогерметизація проколів до 4 мм без втрати тиску.
- **Run Flat (r-f)** — їзда до 80 км на спущеній шині.

## Популярні лінійки

### Літні
- **P Zero** — еталон спортивних шин. Максимальне зчеплення на сухій дорозі.
- **Cinturato P7 (C2)** — комфорт та ефективність для щоденної їзди.
- **Powergy** — середній сегмент Pirelli, гарний баланс характеристик.

### Зимові
- **Sottozero 3** — преміум зимова шина для спортивних та преміум авто.
- **Ice Zero 2** — шиповані, для суворих зим.
- **Ice Zero FR** — фрикційні, для помірних зим.

### Для SUV
- **Scorpion Verde** — екологічні шосейні для кросоверів.
- **Scorpion All Terrain Plus** — для легкого бездоріжжя.

## Сильні сторони

- Найкращі спортивні шини (P Zero — вибір суперкарів).
- Технології Формули-1 у цивільних шинах.
- Преміальна якість та імідж.
- Широкий асортимент для спортивних та преміум автомобілів.

## Слабкі сторони

- Найвища ціна серед преміуму (особливо P Zero та Sottozero).
- Ресурс спортивних моделей нижчий за конкурентів.
- Комфортні моделі (Cinturato) поступаються Michelin Primacy та Continental PremiumContact.
- Менший вибір бюджетних розмірів.

## Для кого?

Власники спортивних та преміум автомобілів. Ідеальні для BMW M, Mercedes-AMG, Porsche, Audi RS. Якщо ваш автомобіль поставлявся з заводу на Pirelli — це найкращий вибір для збереження характеристик.
