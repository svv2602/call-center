# BFGoodrich та Kleber — суббренди Michelin

## BFGoodrich

### Загальна інформація

- **Країна:** США (з 1870), належить Michelin з 1990.
- **Позиція на ринку:** Середній+ сегмент, спеціалізація на позашляховиках та SUV.
- **Ціновий сегмент:** Середній (на 15-25% дешевше Michelin).
- **Конвеєрні поставки:** Jeep, Ford (пікапи).

### Ключові технології

- **CoreGard** — посилені бічні стінки для захисту від порізів та проколів на бездоріжжі.
- **Terrain-T/A** — протекторний малюнок з агресивними грунтозачепами для максимальної прохідності.
- **ETEC** — технологія рівномірного зносу для збільшення ресурсу.

### Популярні лінійки

#### Літні / Позашляхові
- **Advantage** — шосейна комфортна шина для кросоверів та седанів. Тихий хід, хороше зчеплення на мокрому.
- **Trail-Terrain T/A** — всесезонна для м'якого бездоріжжя та щоденної їзди, 3PMSF.

#### Всесезонні / AT
- **All-Terrain T/A KO2** — легенда серед AT-шин. Для серйозного бездоріжжя та щоденних поїздок. 3PMSF, ресурс до 80 000 км.

#### Зимові
- **Winter T/A KSI** — фрикційна зимова для SUV та пікапів, впевнена на снігу.

### Сильні сторони

- Еталон для off-road та AT-шин на українському ринку.
- Посилені бічні стінки — менше проколів на ґрунтових дорогах.
- Великий ресурс (60 000–80 000 км для AT).
- Привабливий агресивний дизайн протектора.

### Слабкі сторони

- Обмежений вибір для легкових авто (фокус на SUV/пікапи).
- Рівень комфорту та шуму поступається чисто шосейним преміум-шинам.

### Для кого?

Власники кросоверів та позашляховиків, які поєднують місто та ґрунтові дороги. Мисливці, рибалки, любителі подорожей.

## Kleber

### Загальна інформація

- **Країна:** Франція (з 1911), належить Michelin.
- **Позиція на ринку:** Бюджет-середній сегмент, "розумна економія".
- **Ціновий сегмент:** Бюджет-середній (на 30-40% дешевше Michelin).
- **Виробництво:** Заводи Michelin у Європі.

### Ключові технології

- Використовує спрощені версії технологій Michelin.
- **StabiliGrip** — стабільне зчеплення на мокрій та сухій дорозі.
- Рецептури гумових сумішей на базі досліджень Michelin.

### Популярні лінійки

#### Літні
- **Dynaxer HP4** — комфортна шина для міста, тихий хід, хороше гальмування на мокрому.
- **Dynaxer UHP** — спортивніша версія, для більш активного водіння.

#### Зимові
- **Krisalp HP3** — фрикційна зимова з хорошим зчепленням на снігу та мокрому покритті.

#### Всесезонні
- **Quadraxer 3** — всесезонна з маркуванням 3PMSF, непоганий компроміс.

### Сильні сторони

- Технології Michelin за меншу ціну.
- Виробляються на європейських заводах Michelin — стабільна якість.
- EU Label рейтинги на рівні B-C.

### Слабкі сторони

- Менший ресурс порівняно з Michelin (на 15-20%).
- Обмежений вибір розмірів для великих діаметрів (R19+).
- Маловідомий бренд в Україні — мала впізнаваність.

### Для кого?

Водії, які хочуть європейську якість за помірну ціну. Гарний вибір для сімейних авто з пробігом до 15 000 км/рік.
